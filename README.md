# entregable2_vue
Actividad para practicar Vue Router and Store en el aplicativo de peliculas con vue

## Pasos para correr el aplicactivo

- npm i
- npm install -D sass sass-loader

Se instala gestor de estados vue pinia
- npm i pinia
- npm i vue-router
